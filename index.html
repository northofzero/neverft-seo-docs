    üìä Publishing Content Updates¶

    Step-by-Step Publishing Process:

    1. Content Management
    2. Navigate to Sanity Studio
    3. Create new content or modify existing entries (countries, cities, venues, blog posts, etc.)

    4. Save your changes as drafts

    5. Publish Changes

    6. Review your content for accuracy

    7. Click the Publish button to make content live in the CMS

    8. Deploy to Website

    9. In the Sanity Studio sidebar, select "Revalidate Pages"
    10. Click "Revalidate All Pages" to sync CMS data with the live website
    11. This updates all published content across the platform and rebuilds affected pages

    üí° Important Notes: - Changes appear on the live site within 1-2 minutes after revalidation - All country-specific pages are automatically updated when global content changes - Use the preview feature to review content before publishing
